Grid System

Using Flexbox and Grid system, both with a standard 12 column system. No matter the size of the browser, each of these columns will always have an equal width.

Class Screen Support
.s{number 1-12} > 0px
.m{number 1-12} >= 768px
.l{number 1-12} >= 992px
.xl{number 1-12} >= 1200px

Using ".grid" class to apply grid layout system.
