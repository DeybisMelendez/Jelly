Colors

Color palette based on Material Design base colors. Each color has a class. Example:

.red

.pink

.purple

.deep-purple

.indigo

.blue

.light-blue

.cyan

.teal

.green

.light-green

.lime

.yellow

.amber

.orange

.deep-orange

.brown

.grey

.blue-grey

.black

.white

Add "-bg" to change background color:

.red-bg

.pink-bg

.purple-bg

.deep-purple-bg

.indigo-bg

.blue-bg

.light-blue-bg

.cyan-bg

.teal-bg

.green-bg

.light-green-bg

.lime-bg

.yellow-bg

.amber-bg

.orange-bg

.deep-orange-bg

.brown-bg

.grey-bg

.blue-grey-bg

.black-bg

.white-bg

Colors has variations lighten and darken: "-lighten-{number}" and "-darken-{number}" from 1 to 3:

.red-darken-3-bg

.red-darken-2-bg

.red-darken-1-bg

.red-bg

.red-lighten-1-bg

.red-lighten-2-bg

.red-lighten-3-bg
